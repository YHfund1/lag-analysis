# Lag Analysis

贷款指标与 10 年期国债收益率的滞后相关性分析。

## 文件

- `index.html`：可离线打开的交互网页，包含 0-20 期滑轨、指标选择、相关系数和双轴折线图。
- `analyze_credit_yield.py`：从本地 Excel 源文件重新生成汇总 Excel 和交互 HTML 的脚本。
- `贷款指标_vs_10年国债收益率_相关性汇总.xlsx`：本次运行生成的相关性汇总和明细表。

## 分析口径

- 样本从 `2015-01-01` 开始。
- 目标变量为 `中债国债到期收益率:10年`。
- 只分析源 Excel 中实际存在的候选因子列；缺失的候选因子会跳过，不会自动补算或改名。
- 滞后定义为：因子领先 10 年国债收益率 x 个月，即用 `t-x` 期因子对应 `t` 期收益率。
- 相关性使用 Pearson 相关系数。
